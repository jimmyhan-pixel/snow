# SilverTask Website

This project combines the SilverTask pages into a runnable website using Vite.

## Project Structure

- `index.html`: Landing/Address page (formerly page1)
- `providers.html`: List of providers (formerly page2)
- `provider-detail.html`: Provider details (formerly page3)

## How to Run

1. Open this folder in your terminal:
   ```bash
   cd website
   ```

2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open the link shown in the terminal (usually `http://localhost:5173`).

## Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist` folder.
